# JobSearcher
A tool that automates the job application process

## Old Workflow
1. Search seek for relevant job posting
2. Add job description to spreadsheet
3. Generate coverletter by passing job description to a LLM
4. Upload CV and Coverletter to seek
5. Click applied on spreadsheet
6. Search emails for rejections 
7. Record rejections on spreadsheet

## Tools for updated workflow
1. Automated job searching and description scraping
- Playwright

2. Updating spreadsheet with job info
- Google app script
- Google sheet api call
- Shift to excel or opensource alternative

3. Generate coverletter
- Ollama
- OpenAI api

4. Upload CV and Coverletter
- Playwright

5. Update spreadsheet
- Google app script
- Google sheet api call
- Shift to excel or opensource alternative

6. Look for rejections
- Gmail API
- Outlook API
- New email client designed for robots

7. Update spreadsheet
- Google app script
- Google sheet api call
- Shift to excel or opensource alternative
